# Issue: 開催しやすさスコアとテーマ状態遷移の導入

## 概要
企画の意思決定を加速するため、テーマ詳細に「開催しやすさスコア」を表示し、テーマの状態遷移（検討→確定→開催済）を明確化する。

## 背景
現在のMVPは投票・参加表明・参加可能時間の集計を提供しているが、主催者が意思決定に必要な情報を素早く判断できる指標が不足している。

## ブランチ名
`feature/decision-score-and-status-flow`

## 受け入れ要件（Acceptance Criteria）
- テーマ詳細ページに「開催しやすさスコア」が表示される。
- スコアは「投票数」「参加表明率」「参加可能時間の偏り」を合成した計算式に基づく。
- テーマの状態が「検討」「確定」「開催済」として表示され、状態変更が可能である。
- 「確定」時に外部イベントURL（converted_event_url）が登録可能である。
- 「開催済」になったテーマは一覧上で区別して表示される。
- 既存の投票/参加表明/参加可能時間機能に影響がないことが確認できる。
